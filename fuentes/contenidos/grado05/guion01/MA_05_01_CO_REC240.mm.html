Conjuntos

  • se concretan en
    • Concepto
      • + - se define como
        • Colección de elementos con características en común.
    • Representación
      • por medio de
        • Diagramas de Venn
          • consiste en
            • Encerrar los elementos con una línea cerrada.
        • Llaves
          • consiste en
            • Escribir los elementos entre llaves.
    • Determinación
      • + - se escriben
        • Por extensión
          • si y solo si
            • Todos sus elementos se escriben entre llaves
        • Por comprensión
          • si
            • Se escribe entre llaves una característica que identifica todos los elementos del cionjunto
    • Operaciones
      • se identifica
        • La unión
          • para los conjuntos A y B
            • {x/x pertenece a A o x pertenece a B}
        • La intersección
          • para los conjuntos A y B    
            • {x/x pertenece a A y x pertenece a B}
        • La diferencia
          • para los conjuntos A y B
            • {x/x pertenece a A y x no pertenece a B}
        • La diferencia simétrica
          • para los conjuntos A y B
            • {x/x pertenece a (A - B) o x pertenece a (B - A)}
        • El complemento
          • para el conjunto A
            • {x/x pertenece al conjunto universal y x no pertenece a A}
    • Clases
      • Finito
        • Aquel en el que se pueden contar todos sus elementos
      • Infinito
        • Conjunto en el que el proceso de contar sus elementos no termina
      • Unitario
        • Conjunto que tiene un solo elemento
      • Vacío
        • Conjunto que no tiene elementos
      • Universal
        • Conjunto que se toma como referencia para formar otros subconjuntos
    • Relaciones
      • entre elemento y conjunto
        • Pertenencia
      • entre conjunto y conjunto
        • Contenencia
          • Un conjunto es subconjunto del otro
        • Disjunción
          • Los conjuntos no tienen elementos en común
        • Igualdad
          • Los conjuntos tienen los mismos elementos
        • Intersecantes
          • Los conjuntos tienen elementos comunes
